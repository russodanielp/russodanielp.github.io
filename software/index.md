## The Chemical _In vitro_-_In vivo_ Profiling (CIIPro) portal

<center>
<img src="/images/ciipro.png" width="650"/>
</center>
<br>
Located at [ciipro.rutgers.edu](ciipro.rutgers.edu), CIIPro is large scale cheminformatics portal designed to profile chemicals based on their __in vitro__ biological assay responses.  Please read the full details of the portal in the journal [__Bioinformatics__](https://academic.oup.com/bioinformatics/article/33/3/464/2617569).

## PubChemQSAR 

PubChemQSAR is a Python library for building [quantitative structure-activity relationships](https://en.wikipedia.org/wiki/Quantitative_structure%E2%80%93activity_relationship) on-the-fly from any PubChem biological assay.  PubChemQSAR is an on going project and the [source code is available on github](https://github.com/russodanielp/pubchemqsar).

    Please visit my github page for a full listing of my software projects.

    The Chemical In vitro-In vivo Profiling (CIIPro) portal


    Located at ciipro.rutgers.edu, CIIPro is large scale cheminformatics portal designed to profile chemicals based on their in vitro biological assay responses. Please read the full details of the portal in the journal Bioinformatics.

    PubChemQSAR

    PubChemQSAR is a Python library for building quantitative structure-activity relationships on-the-fly from any PubChem biological assay. PubChemQSAR is an on going project and the source code is available on github.
